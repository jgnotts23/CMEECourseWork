### Description ###

This directory and its subsidaries contains the code created in week 9 of the Computational Methods in Ecology and Evolution MSc at Imperial College London. The purpose of this week was to learn how to use the HPC and about neutral theory and fractals.


### Dependencies ###

Everything in this project was created and run with the ubuntu 16.04 OS. R scripts were run in R 3.2.3.
