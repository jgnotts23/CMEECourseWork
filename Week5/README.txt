### Description ###

This directory and its subsidaries contains the code created in week 5 of the Computational Methods in Ecology and Evolution MSc at Imperial College London. The purpose of this week was to learn GIS techniques and most of the work was carried out on a different college PC.


### Dependencies ###

Everything in this project was created and run with the ubuntu 16.04 OS.
