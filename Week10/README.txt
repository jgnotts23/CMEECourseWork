### Description ###

This directory and its subsidaries contains the code created in week 10 of the Computational Methods in Ecology and Evolution MSc at Imperial College London. The purpose of this week was to learn C coding.


### Dependencies ###

Everything in this project was created and run with the ubuntu 16.04 OS. C code was compiled with gcc 5.4.0
